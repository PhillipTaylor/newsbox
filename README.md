
# Newsbox

This is a command line news browser written by ChatGPT:

https://chatgpt.com/share/69837910-c390-8009-a808-20bd0f36b3fb

To run this app use:

```bash
cargo run
```

press `r` to populate the feed, navigate up and down with `j` and `k` keys and press `o` to open the article in your browser
